# Code of Conduct

## The short version

Be kind. Be useful. Be precise.

## The longer version

This community is a working environment, not a chat room. Treat people the way you'd want to be treated by a good colleague.

**What's expected:**
- Engage with the work, not the person.
- Be specific. "This is wrong because of X on line Y" beats "this is bad."
- Disagree politely and from substance.
- Apologise when you mess up. We all do.

**What's not OK:**
- Personal attacks, harassment, slurs, doxxing.
- Sexualised language or imagery in any project space.
- Pile-on dynamics. One critique is enough.
- Bad-faith engagement (concern trolling, sealioning, performative outrage).

## Enforcement

Maintainers can lock threads, edit comments, remove contributions, or ban users for violations. We aim to act proportionately and explain why when we do.

If you've been on the receiving end of behaviour that violates this code, email **conduct@erp.ai** with the relevant URLs and we'll respond within 72 hours.
